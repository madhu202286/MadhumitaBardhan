Airbnb

OVERVIEW

In this project, I have gone through the yearly data of airbmb for the year 2016. I sourced this dataset from Kaggle. My main focus was to get familiar with Tableau. I created appropriate visualizations and later compiled them together to form an insightful dashboard.

RESOURCES & TOOLS

1. Kaggle
2. Tableau

DATA CLEANING

For this dataset, I first did the cleaning and refining part in excel, then connected the csv text file with tableau.

DATA ANALYSIS

I created various types of charts like bar chart, map, line graph along with colorful marks to see the yearly revenue of airbnb and how the prices of rooms vary by location and number of bedrooms.

DATA VISUALIZATION

Next, I created a dashboard in tableau from the charts. The zipcodes here act as filter buttons and provides a dynamic visualization.
