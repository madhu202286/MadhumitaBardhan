Bike Sales

OVERVIEW

In this project, I have gone through the sales data of Motor Bikes. My main goal was to learn about Data Cleaning, Data Refining, making Pivot Charts & Pivot Tables and finally creating a insightful sales dashboard within Excel. I focused on doing every task in Excel so as to learn the important techniques regarding cleaning and preparing the raw data, analyzing the data and creating communicable data visualizations.

RESOURCES & TOOLS

1. Kaggle
2. MS Excel

DATA CLEANING

I got rid of duplicate and null values. Then one by one I checked each column and refined them. For the Marital status and Gender column, I replaced the text appropriately. Then for the Income column, I removed the currency sign. To correct any other typos present, I applied text filter, but no more typos were there in this data set. Next I corrected the range of Commute distance.

DATA ANALYSIS

I created pivot tables and pivot charts to see how purchase of bikes are being affected by income, gender, age and commute distance of customer.

DATA VISUALIZATION

Next, I went ahead to create a dashboard in excel from the pivot charts I created. To add filter buttons to the charts, I created slicers and connected them to all the charts. This made the dasboard more insightful.
